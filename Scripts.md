### Bloque  diff

Para mostrar modificaciones de codigo. 
```diff
- texto en rojo
+ texto en verde
! texto en raranja
# texto en gris
@@ texto en morado y negrita@@
```

Por ejemplo, cuando codificamos una linea
````
```diff
public class Hola
{
   public static void Main()
   {
-      System.Console.WriteLine("Hello, World!");
+      System.Console.WriteLine("Bye!");
   }
}
```
````

```diff

{
```
