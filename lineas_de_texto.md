# Cursiva y negrita

```
*cursiva 1*	
_cursiva 2_	
**negrita 1**
__negrita 2__	
***cursiva y negrita 1***	
___cursiva y negrita 2___
```

*cursiva 1*	

_cursiva 2_	

**negrita 1**

__negrita 2__	

***cursiva y negrita 1***	

___cursiva y negrita 2___


# Escape markdown

Para escapar de markdown usamos la barra invertida `\`
```
\*  asterisco
\#  almohadilla
\+  símbolo de suma
\-  guión
```

\*  asterisco

\#  almohadilla

\+  símbolo de suma

\-  guión


# Comentario en markdown

```
[//]: # (Este es un comentario)
```
El comentario esta entre este parrafo y

[//]: # (Este es un comentario)

este otro parrafo. (No debe aparecer el comentario)
